# Login Page UI Project

A simple login page built using **HTML and CSS**, focused on creating a clean and responsive user interface.

## Features
- Basic login page layout  
- Responsive design  
- Custom CSS styling  
- Logo integration  

## Project Files
- `index.html` – Main login page structure  
- `style.css` – Styling for the login page  
- `klnce logo.png` – Logo asset  

## Purpose
This project was created to practice front-end web development fundamentals, including HTML structure, CSS styling, and basic UI design.
